A [Arduino](https://www.arduino.cc) [Leonardo](https://store.arduino.cc/arduino-leonardo-with-headers) like PCB with 65 LED's and 5 Buttons.  
MCU - ATmega32U4-AU (16MHz)  
LED Driver - Max7219CWG  

Name            | Pin
--------------- | ---------------------
Button 1        | D0  
Button 2        | D1  
Button 3        | D2  
Button 4        | D3  
Button 5        | D4  
Max7219CWG LOAD | D10  
Max7219CWG CLK  | D11  
Max7219CWG DIN  | D12  
LED 1           | D13  
LED 2 to LED 65 | Max7219CWG
